Права доступа!!

"-" обычный файл
"d" каталог
"I" символьная ссылка
"s" сокет

"r" - read
"w" - write
"x" - exec исполнять

например напишем ls -la чтоб узнать права и увидим

##drwxr-xr-x

первая буква d - ощначет что это каталог
затем разбиваем строку по 3 и видим
[rwx] + [r-x] + [r-x]
rwx     права доступа для sudo (означает что судо пользователь может читать, писать, исполнять)
r-x     права доступа для user (пользователя) (может читать, -, исполнять)
r-x     права доступа для остальный
